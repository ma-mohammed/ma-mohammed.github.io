Projects

  • 2021.01 - 2024.12

    Advanced model integrating trip chaining, passenger typology, and spatial error correction to infer complete trip patterns including transfers in regional transit systems.

    • Scalable for regional agencies
  • 2023.01 - 2024.12
    Crash Typology Analysis
    Machine learning framework using UMAP and Gaussian Mixture Modeling to classify census tracts into distinct crash typologies based on historical patterns.
    • Analysis of 2,480 census tracts
    • UMAP dimensionality reduction
    • Spatial visualization tools
